# 웰빙이너스 (wellbeingin.us)

주식회사 웰빙이너스 공식 홈페이지 — AI 기반 생활·업무 서비스 허브.

## 스택

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Pretendard

## 로컬 개발

```bash
npm install
npm run dev
```

http://localhost:3000
